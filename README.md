# Credit-Risk-Prediction
Advanced credit risk (loan default) prediction using PyTorch neural network with extensive feature engineering, Focal Loss for imbalance, AdamW + OneCycleLR scheduling. Includes ROC-AUC, PR curve, model comparison (RF, GB), and SHAP explainability. Achieved 93.11% accuracy, 0.9169 ROC-AUC on test set. 

The paper can be accessed using the DOI: 10.21203/rs.3.rs-8390678/v1 (https://doi.org/10.21203/rs.3.rs-8390678/v1).
